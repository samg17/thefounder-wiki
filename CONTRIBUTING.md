# Contribuindo

Pull requests são bem-vindos. Este repositório é um template compartilhado — o conteúdo-semente deve ser útil para qualquer founder, independente do mercado específico.

---

## O que contribuir

- Novas páginas de frameworks comprovados (com experiência prática, não só teoria)
- Guardrails baseados em erros reais observados em founders
- Conceitos com definições mais claras ou mais acionáveis do que as existentes
- Correções e atualizações de conteúdo desatualizado

## O que não contribuir

- Conteúdo promocional ou de empresa específica
- Frameworks sem evidência de uso prático
- Conteúdo de entidades específicas (mercados, players) — esse conteúdo é gerado pelo usuário com base em suas próprias fontes

---

## Convenções

Toda página deve seguir o template em `templates/pagina-padrao.md`:

1. **Frontmatter completo** — title, category, tags, updated, status, related
2. **Linguagem acionável** — escreva para um founder que precisa decidir hoje
3. **Seção de erros comuns** — com formato ❌/✅
4. **Links internos** — use `[[nome-da-pagina]]` para conectar ao resto do wiki
5. **Status inicial:** `draft` — o mantenedor revisa e promove para `reviewed`

---

## Processo

1. Fork o repositório
2. Crie ou edite a página seguindo as convenções
3. Abra um PR com descrição breve do que foi adicionado e por quê
4. O PR é revisado quanto a qualidade, acionabilidade e consistência com o resto do wiki
